Features

Everything you need to feed documents into an LLM

Distill handles the hard parts – structure extraction, OCR, vision captioning, and token efficiency – so your pipeline doesn’t have to.

📄

Structure preserved

Headings, tables, lists, bold/italic, code blocks, hyperlinks, and speaker notes all survive the conversion.

📉

Token-efficient output

50–70% fewer tokens than naive extraction. Every conversion reports a quality score so you know what was preserved.

🔍

Scanned PDF OCR

Auto-detects image-only PDFs and runs layout-aware OCR via docling or Tesseract. No manual pre-processing needed.

🖼️

Vision captioning

Describe images embedded in documents using OpenAI, Anthropic, or Ollama. Works locally with no cloud required.

⚡

Streaming API

Yield one Markdown chunk per section for real-time LLM pipelines without buffering the whole document.

🌐

Web UI + REST API

Local browser interface and POST /api/convert endpoint included. No account, no cloud, no subscription.

Quickstart

Up and running in two lines

Install the core library, point it at a file, get clean Markdown back.

example.py 
from distill import convert

result = convert("report.docx")

# Clean Markdown string
print(result.markdown)
# 0.0–1.0 structure quality
print(result.quality_score)
# Document title, author, etc.
print(result.metadata.title)
# Any conversion warnings
print(result.warnings)
streaming.py 
from distill import convert_stream

# Stream one chunk per section
for chunk in convert_stream("report.docx"):
    print(chunk)

# REST API – no Python needed
# curl -X POST localhost:7860/api/convert \
# -F "file=@report.pdf"

# Web UI
$ distill-app
# opens http://localhost:7860
